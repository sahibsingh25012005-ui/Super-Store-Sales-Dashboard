# 📊 Super Store Sales Dashboard

An interactive Power BI dashboard built to analyze sales performance, profitability, customer segments, and regional trends for a retail superstore.

## Dashboard Highlights
- Total Sales: **$1.57M**
- Total Profit: **$175.26K**
- Units Sold: **22.32K**
- Average Delivery Time: **3.93 Days**

## Key Insights
- Consumer segment generates the highest sales.
- Office Supplies is the top-performing category.
- Phones lead all sub-categories in sales.
- Standard Class is the most used shipping mode.
- Sales and profit peak during the last quarter of the year.

## Data Source
-Sample Superstore dataset used for educational and analytical purposes.

## Tools Used
- Power BI
- Power Query
- DAX
- Excel/CSV

## Features
- Interactive Region & Year Filters
- Sales and Profit Trend Analysis
- Category & Sub-Category Performance
- Regional Sales Map
- Customer Segment Analysis
- Shipping Mode Analysis
